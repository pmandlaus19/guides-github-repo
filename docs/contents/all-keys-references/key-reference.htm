You can designate up to eight servers for each AAA service and up to eight global servers. Services are performed by the corresponding TACACS+ component on an AAA server. If AAA servers are not configured or are inaccessible, the global server list is used.
Server priority is configurable (1= highest) and determines the order that clients contact TACACS+ servers. As you add or remove servers, the priority value is incremented or decremented automatically to prevent gaps in server priority. You can search for servers by priority or cached value (last accessed). The AAA services also maintain statistics that you can retrieve or clear.
