Add a TACACS+ authorization server
Note:
You can add up to eight authorization servers.
tacacs authorization add server <IP address or host name> {[priority <Number: 1...8>] [tcp-port <Number: 1...65535>]}
Note:
The default value of the [tcp-port] attribute is 49. Ensure that the TCP port to be used is not already reserved.
Set the administrative state of an authorization server
tacacs authorization <disable | enable> server <IP address or host name>
Retrieve summary information or statistics for configured authorization servers
tacacs authorization show [statistics]
Clear statistics for the specified authorization server
tacacs authorization clear server <IP address or host name> statistics
