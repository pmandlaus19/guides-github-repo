Add a server to the global server list
Note:
You can add up to eight global servers.
tacacs add server <IP address or host name> {[priority <Number:1...8>] [tcp-port <Number:1...65535>]}
Note:
The default value of the [tcp-port] attribute is 49. Ensure that the TCP port to be used is not already reserved.
Set the administrative state of a global server
tacacs <disable | enable> server <IP address or host name>
Retrieve statistics for configured global servers
tacacs show [statistics]
Clear statistics for the specified global server
tacacs clear server <IP address or host name> statistics
